# FILE60 // VISUAL TERMINAL

## Overview

This is a single-page HTML application for temporary file storage. It allows users to upload images and videos, which are automatically deleted after 60 minutes.

The application features a "Brutalist" / "Industrial" UI design with:
- Drag and drop file upload
- Visual expiration timers
- Local storage persistence
- Responsive layout

## Usage

Simply open `index.html` in any modern web browser.

No build process or server installation is required.

## Technologies

- **HTML5**
- **Tailwind CSS** (via CDN)
- **Vanilla JavaScript**
- **TmpFiles.org API** (for backend storage)
